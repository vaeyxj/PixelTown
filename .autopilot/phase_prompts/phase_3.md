# Phase 3: UI/UX 游戏化设计

你正在开发 PixelTown — AI 教育公司像素风办公世界。
读取 CLAUDE.md 和 .autopilot/TASK_NOTES.md 了解上下文。

## 背景
渲染引擎已升级为 sprite-based + 视差 + 粒子。现在要把网页 UI 做成 RPG 游戏风格，让整个体验像在玩一款精致的像素 RPG。

## 核心理念
这是一个公司员工展示世界，不是真的游戏。但 UI 要用 RPG 风格包装：
- 角色详情 = RPG 角色面板
- 部门 = 阵营/公会
- 属性 = RPG 五维图
- 技能 = 技能树
- 成长记录 = 冒险日志

## 任务

### 3a. 登录页重做 (`LoginScreen.tsx`)
- 全屏像素风背景（使用 Phase 1 生成的背景图）
- 标题 "PIXEL TOWN" 打字机逐字显示效果
- 副标题 "AI教育 · 像素办公世界" 渐入
- 像素风发光按钮 "进入世界 ▶"
- 进入时镜头推进动画（从楼外到室内）
- 背景有缓慢飘动的光斑粒子

### 3b. HUD 系统重做 (`components/HUD.tsx`)
- **顶部状态栏**: RPG 风格像素边框
  - 像素翻页钟（非纯文字，要有翻页动画感）
  - 在线人数用像素风 LED 数字显示
  - 各状态人数用小图标 + 数字
- **底部工具栏**: RPG 快捷栏风格
  - 6 个格子：🗺️地图 / 👥员工图鉴 / 📊统计 / 🔍搜索 / ⚙️设置 / 🔊音效
  - 每个格子像素风边框 + 悬停高亮 + 点击弹出对应面板
- **小地图**: 右下角，像素风雷达样式，有扫描线动画

### 3c. 角色详情面板 (`components/CharacterPanel.tsx`)
点击任何角色弹出 RPG 风格面板：
- **布局**: 像素风窗口边框，可拖拽，点击外部或 ✕ 关闭
- **头像区**: 左侧大像素头像 + 名称 + 英文名 + 职级徽章
- **属性区**: 五维雷达图（Canvas 绘制，像素风格线条）
  - 教学力 / 研发力 / 创造力 / 影响力 / 协作力
  - 数值显示 + 雷达图可视化
- **技能区**: 技能列表，每个技能有图标 + 名称 + 等级条（像素风进度条，1-10 格）
- **简介区**: 个人简介文本 + 状态签名
- **博客区**: 折叠列表，点击展开摘要，显示标签
- **成长时间线**: 纵向时间轴，像素风节点 + 里程碑事件
- **标签**: 底部个人标签，像素风徽章样式

### 3d. 员工图鉴 (`components/EmployeeGrid.tsx`)
- 全屏覆盖面板（像素风窗口）
- 网格卡片布局：每张卡片像素风卡牌样式
  - 头像 + 名字 + 部门色条 + 职级
  - 悬停显示 motto
- 顶部筛选栏：按部门过滤（彩色标签按钮）
- 搜索框：像素风输入框
- 按属性排序：点击表头切换排序维度
- 点击卡片跳转到角色详情

### 3e. 统计仪表盘 (`components/Dashboard.tsx`)
- 像素风面板，网格布局
- 实时数据卡片（用像素翻牌器效果）：
  - 当前在线人数
  - 各状态分布（工作/会议/午休...）
  - 部门人数分布
- 像素风柱状图（用 div 堆叠实现，不引入图表库）
- 像素风进度条展示各部门平均属性

### 3f. 页面过渡和微交互
- 面板打开/关闭：像素风卷帘动画（从中间展开）
- 按钮 hover：像素弹跳效果（上移2px + 阴影）
- 通知/toast：像素风气泡从底部弹出
- 标签页切换：像素风滑动过渡
- 所有数字变化：像素风翻牌器动画

### 3g. 像素风 CSS 设计系统
创建 `src/styles/pixel-ui.css`:
- 像素风字体：使用像素字体或 monospace + image-rendering: pixelated
- 通用像素边框类：.pixel-border（用 box-shadow 模拟像素边框）
- 按钮样式：.pixel-btn / .pixel-btn-primary / .pixel-btn-danger
- 输入框样式：.pixel-input
- 面板样式：.pixel-panel / .pixel-window
- 进度条：.pixel-progress
- 徽章：.pixel-badge
- 动画：@keyframes pixel-bounce, pixel-fade-in, pixel-slide-up

## 规则
- 所有 UI 用纯 CSS + React 实现，不引入 UI 框架
- 像素风一致性：所有边框、阴影、动画都要保持 8-bit 风格
- 面板数据从 src/data/employees.ts 读取
- commit 格式: `feat(phase-3): 描述`
- 更新 .autopilot/TASK_NOTES.md
